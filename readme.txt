# Proyecto de filtrado de productos

Este proyecto es un ejemplo de cómo filtrar una base de datos de productos utilizando JavaScript y HTML.

## Tabla de contenido

- [Instalación](#instalación)
- [Uso](#uso)
- [Estructura del proyecto](#estructura-del-proyecto)
- [Licencia](#licencia)

## Instalación

Para instalar el proyecto, simplemente descarga el archivo ZIP y extrae su contenido en una carpeta en tu ordenador.

## Uso

Para utilizar el proyecto, abre el archivo index.html en tu navegador web. Luego, utiliza el formulario para filtrar los productos por marca, color, airbag y precio. Cuando hagas clic en el botón de filtrar, se mostrarán los productos que cumplan los filtros seleccionados.
